## BarraNeutralizer

BARRA-style 2-stage factor neutralization.

Full BARRA process:
1. Estimate factor loading matrix B via time-series regression
   R_i,t = Σ(β_i,k × F_k,t) + ε_i,t

2. Neutralize alpha via cross-sectional regression
   α_raw = B × γ + α_pure
   Result: α_pure (residual) = pure alpha with factor exposure removed

This operator maintains rolling history for loading estimation.

Example:
    graph.add_node("neutral_alpha", BarraNeutralizer(
        alpha_input=Input("momentum", timeframe="1d", lookback=1),
        returns_input=Input("returns", timeframe="1d", lookback=60),
        factor_inputs=[
            Input("volatility", timeframe="1d", lookback=60),
            Input("size", timeframe="1d", lookback=60),
        ],
        factor_names=["VOL", "SIZE"],
        estimation_window=60,
    ))
