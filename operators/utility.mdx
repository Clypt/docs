## IntervalGate

Time-based gate that opens on timeframe boundaries.

Outputs 1.0 when current timestamp is on the specified interval boundary,
0.0 otherwise. Use this to control execution frequency of operators
that don't have internal data to aggregate (e.g., WebSearch, LLM).

**Role**: `UNKNOWN` | **Ephemeral**: No

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `trigger_input` | `Input` | Required |  |
| `interval` | `str` | `'5m'` |  |
| `output_timeframe` | `Optional[str]` | `None` | Output timeframe (defaults to trigger input timeframe) |
