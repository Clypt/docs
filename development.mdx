## Local Development

Edit your ClyptQ documentation locally and preview changes in real-time.

### Prerequisites

- Node.js 16+ and npm
- Git
- Text editor (VS Code recommended)

### Setup Steps

<Steps>
  <Step title="Install Mintlify CLI">
    Install the Mintlify CLI globally:

    ```bash
    npm i -g mintlify
    ```
  </Step>

  <Step title="Clone the repository">
    Clone the docs repository:

    ```bash
    git clone https://github.com/clypt/docs.git
    cd docs
    ```
  </Step>

  <Step title="Start development server">
    Run the development server:

    ```bash
    mintlify dev
    ```

    Your docs will be available at `http://localhost:3000`

    <Info>
    The preview automatically updates as you edit files.
    </Info>
  </Step>
</Steps>

### Writing Documentation

All documentation is written in MDX (Markdown + React components):

```mdx
---
title: "Page Title"
description: "Page description"
icon: "rocket"
---

## Section Title

Content goes here...

<Tip>
Use Mintlify components for rich content!
</Tip>
```

## Publishing Changes

<Steps>
  <Step title="Test locally">
    Verify your changes look correct:

    ```bash
    mintlify dev
    ```
  </Step>

  <Step title="Commit changes">
    Commit your changes:

    ```bash
    git add .
    git commit -m "Update documentation"
    ```
  </Step>

  <Step title="Push to GitHub">
    Push to the repository:

    ```bash
    git push origin main
    ```

    <Success>
    Your changes will be automatically deployed!
    </Success>
  </Step>
</Steps>
